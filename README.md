# NewsApp一个练手的新闻App

#### * 数据来源: 天行数据APi 

#### * 整体架构基于 MVP + Retrofit + RxJava
 
#### * 第三方库 : Glide、Gson、butterknfie、Banner等
 
####  *主要采用 SwipeRefreshLayout + RecyclerView 实现多种布局
